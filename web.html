    Apa itu HTML:

  • Sebuah bahasa markup yang digunakan untuk membuat sebuah halaman web dan menampilkan berbagai informasi di dalam sebuah browser
  • HTML merupakan sebuah kode-kode tag yang menginstruksikan browser untuk menghasilkan tampilan sesuai dengan apa yang diinginkan
  • HTML saat ini merupakan standar internet yang didefinisikan dan dikendalikan penggunaannya oleh World Wide Web Consortium (W3C)

    Ciri-ciri dokumen HTML:

  • Ekstensi file berupa .htm atau .html
  • Non case sensitive
  • Terdiri atas tag-tag pembuka dan penutup (walaupun ada beberapa tag yang tidak memiliki penutup)
  • Tag-tag saling berpasangan dan bersarang

    Struktur HTML:

  • Dokumen information
  • Dokumen header
  • Dokumen body

    Document Header:

  • Page title
  • Scripting
  • Style
  • Meta

    Document Body:

Bagian dari dokumen web yang akan ditampilkan ke user.
Bagian ini memuat:

  • Teks dan gambar
  • Link
  • Server Side Script (PHP,ASP,JSP)
  • Multimedia and Special Programmed Events (Shokewave, SWFs, Java Applets, Online video)

    TAG HTML

  1. Penulisan Tag
    • Tag pembuka
    • Tag penutup
    • Single tag
  2. Tag Heading
    berfungsi untuk menuliskan judul & sub-judul
  3. Tag Paragraf, HR, dan BR
    • Tag paragraf: membuat paragraf baru, membuat teks berada dalam sebuah paragraf. Tag paragraf bisa memiliki tag penutup, bisa juga tidak.
    • < br/>: break-line (untuk berpindah ke baris selanjutnya)
    • < hr/>: horizontal-line (untuk menambahkan garis horizontal)
  4. Font tag
    untuk mengatur penggunan tulisan dalam halaman web (jenis tulisan, ukuran, warna, dll)
    Tag lain untuk manipulasi font:
    • < br>: tulisan tebal < /br>
    • < i>: tulisan miring < /i>
    • < u>: tulisan bergaris bawah < u/>
    • < sub>: subscript < /sub>
    • < sup>: superscript < /sup>
  5. Image tag
    untuk memuat gambar ke dalam halaman web
    Tag image memiliki beberapa attribute antara lain:
    • Src: lokasi gambar yang akan ditampilkan
    • Width: ukuran lebar gambar
    • Height: ukuran tinggi gambar
    • Alt: deskripsi tentang gambar
    • Tittle: judul gambar
  6. Anchor/ Hyperlink tag
    dipergunakan untuk menghubungkan (linking) teks dan image ke halaman lain atau bagian tertentu dari halaman yang sama dalam satu website atau website yang lain
    contoh:
    • Link ke halaman website lain
    • Link ke file lain dalam satu website

    untuk mengakses bagian tertentu dalam sebuah halaman website, maka bagian halaman tersebut harus diberi tanda.
    contoh:

    • Penanda pada bagian halaman
    • Link ke file yang
  7. Tag komentar
    befungsi sebagai pembungkus dokumen HTML atau komentar, agar tidak terbaca oleh browser.
